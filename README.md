## 🛠️ Featured Projects

### 🔵 Signal — AI-Native Returns Discovery
* **Problem:** E-commerce Returns PMs waste 7–8 hours weekly manually stitching 6+ disconnected data sources just to find actionable friction points.
* **Solution Built:** Architected an on-demand AI discovery tool featuring a three-tier LLM pipeline (RAG query nodes → Master LLM → Judge Node). The architecture was specifically designed to enforce output validation and live faithfulness scoring, preventing hallucinated insights and linking every claim to raw source evidence.
* **Business Impact:** Quantified a $62M–$158M annual business case. Designed to compress discovery time to 2 hours/week, tracking success via the *% of quarterly PRFAQs defined using tool-generated suggestions*.
* → [View Repository](https://github.com/reagleai/signal) | [View Prototype](https://signal.protonaiagents.com/)

### 🟣 Resumatch — AI Resume Tailoring Engine
* **Problem:** High-volume job seekers face a compounding 20–30 minute bottleneck manually rewriting their resume for every single application.
* **Solution Built:** Developed a web application backed by a 9-stage automation pipeline. *Core Product Decision:* Prioritized document safety over raw generation. The system isolates and rewrites only 4 qualitative sections (Summary, Experience, Projects, Skills) while strictly locking all factual data (titles, tenures, companies) to ensure 100% factual accuracy.
* **Business Impact:** Compressed a tedious manual workflow into a ~1 minute automated run, shipping with a secure cloud dashboard so users never lose historical application artifacts.
* → [View Repository](https://github.com/reagleai/auto-resume-optimizer) | [View Prototype](https://resumatch.protonaiagents.com/)

### 🟡 Smart Cart — Instamart Retention Engine
* **Problem:** Validated via primary user research that 64% of "Planner" users abandon the platform for offline local vendors when they forget a single item.
* **Solution Built:** Prototyped a dual-layered, Gemini API-powered contextual suggestion engine triggered at the cart step. Engineered a strict latency constraint (< 5 seconds) utilizing a local rule-based fallback to guarantee the AI check never introduces friction into the checkout flow.
* **Business Impact:** Designed a comprehensive A/B test framework on 125K users, targeting an expected +1.2 items per basket and a 10% lift in repeat order frequency.
* → [View Repository](https://github.com/reagleai/smart-cart-recommendations) | [View Prototype](https://smartcartrecommendations.protonaiagents.com/)

### 🟢 Echo — Human-in-the-Loop LinkedIn Agent
* **Problem:** Consistent professional networking requires 45+ minutes of daily manual discovery and context-switching.
* **Solution Built:** Orchestrated a multi-model pipeline (Tavily search → DeepSeek → Claude 3.5). *Core Product Decision:* Deliberately excluded automated posting after platform research confirmed severe algorithmic penalties. Kept the human firmly in the loop by delivering AI-generated drafts directly to a structured Google Sheet for review.
* **Business Impact:** Automated the heavy lifting of content discovery and drafting while preserving account safety and authentic human review.
* → [View Repository](https://github.com/reagleai/echo) | [View Live Product](https://linkedinengage.protonaiagents.com/)
